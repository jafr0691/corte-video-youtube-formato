    Questions fréquemment posées

    Comment définir la qualité du clip?

    Il suffit de définir la qualité vidéo souhaitée dans le lecteur YouTube. Le clip que vous téléchargez aura la même qualité.

    Quelle est la longueur maximale du clip?

    Pour les clips vidéo, il est à 10 minutes, pour les fichiers audio, il est 1 heure et pour GIFs il est 10 secondes.

    Comment prendre une capture d'écran du film YOUTUBE?

    Réglez le « heure de début » à l'image que vous voulez capturer. L'heure de fin est ignoré, donc il réglé sur une valeur qui permettra le téléchargement.

    Quels types de fichiers que vous produisez?

    Nous essayons de générer des fichiers mp4 pour les vidéos et m4a pour l'audio. Si, pour une entrée donnée, il est impossible que vous pouvez obtenir des Matroska (mkv, MKA) ou Vorbis (ogg). Regardez ici pour trouver des joueurs pour ces types de fichiers.

    Comment télécharger un album complet de YouTube et le diviser en morceaux?

    Utilisez notre site soeur YTalbums.com , qui fait exactement cela. Vous pouvez spécifier une liste des pistes de sorte que chaque chanson a le bon nom et vous pouvez télécharger toutes les chansons dans une archive ZIP.
